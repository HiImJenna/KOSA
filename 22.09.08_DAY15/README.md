## 1. 라이브러리와 프레임워크 ✔
-----------------------------
### 🔔 차이점
- 흐름에 대한 제어 권한이 어디에 있느냐
- 프레임워크는 전체적인 흐름을 자체적으로 가지고 있으며, 프로그래머가 그 안에 필요한 코드를 작성한다. => 제어의 역전이 적용되어있다
- 라이브러리는 사용자가 흐름에 대해 제어를 하며 필요한 상황에 가져다 쓴다.
