## 1. 정규화 ✔
- 1차 정규화 : 모든 속성은 원자값을 가짐, 복수의 속성값을 가지는 속성을 분리
- 2차 정규화 : 주식별자에 종속적이지 않은 속성의 분리, 부분 종속 속성을 분리 - 부분함수종속성
- 3차 정규화 : 속성에 종속적인 속성의 분리, 이전 종속 속성의 분리 - 이행함수종속성
- 보이스-코드 정규화 : 다수의 주식별자 분리
- 4차 정규화 : 속상간의 다중속성 제거 - 다중종속성
