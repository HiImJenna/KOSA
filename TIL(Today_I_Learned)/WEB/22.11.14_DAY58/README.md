### 🔔 항공 시나리오 초안
#### < 나 - 항공 ✈️ >
```
1. 이용자
    - 항공 예약을 원하는 개인 회원, 관리자 회원으로 구분된다.
    - 개인 회원은 인천공항과 제주공항을 오고가는 00사, ㅁㅁ사의 항공기를 조회하고 예약할 수 있다.
    - 개인 회원은 날짜와 운행 시간, 좌석 등급, 티켓 유형(왕복, 편도), 승객 수를 선택하여 원하는 항공권 티켓을 구매할 수 있다.
    - 개인 회원은 항공권 예약 시 승객정보와 회원정보를 동일하게 할 지 여부를 선택할 수 있다.
    - 개인 회원은 마이페이지에서 항공권 예약 내역을 확인할 수 있다.
    - 개인 회원은 항공권 예약을 마친 후 예약 정보가 담긴 e-ticket을 pdf 파일로 확인할 수 있다.
    - 개인 회원은 결제 시 발행된 쿠폰 번호를 입력해 할인을 받을 수 있다.
    - 개인 회원은 고객 센터 게시판을 통해 관리자에게 문의를 남길 수 있다.
    - 개인 회원은 항공권 예약을 취소하고 싶을 시, 관리자에게 취소 요청을 보내야한다.
    
2. 관리자
    - 관리자는 회원이 고객센터에 남긴 문의글에 답글 기능을 통해 글을 남길 수 있다.
    - 관리자는 개인 회원의 항공권 예약 취소 요청을 승인 또는 거절하여 처리할 수 있다.
    
3. 서비스
    - 항공권 예약이 완료될 시 공항 버스, 고속 버스 등의 추가적인 교통 정보를 화면을 통해 제공한다. (단순 정보제공)
    - 항공권 예약이 완료될 시 목적지 공항 근처 맛집을 확인할 수 있도록 (동’s) 맛집 카테고리로 연결해주는 서비스를 제공한다.
```

#### < 승복님 - 숙소 예약 >
```
호텔예약 페이지

- 고객에게 호텔 예약을 가능하게 하는 페이지다.
- 좌측 상단에는 숙박 장소를 선택할 수 있는 위치(강남구, 강서구, 광진구 … 중랑구)를 선택 할 수 있다.
- 우측 상단 위치 아래에는 숙박일을 선택할 수 있는데, 체크인과 체크아웃 날짜를 선택 가능하다.
- 해당 플랫폼에는 사이트에서 제공하는 호텔 리스트가 호텔 등급에 따라 제공된다.
- 호텔리스트의 호텔은 호텔 이름, 호텔 등급, 호텔 사진이 제공된다.
- 호텔 등급은 특1급, 특2급, 1급, 2급, 일반 순으로 나누어진다.

날짜 지정 페이지

- 가운데 달력에서 체크인 날짜와 체크아웃 날짜를 선택 가능하다.
- 좌측 상단에는 선택된 체크인 날짜를 표기하며, 우측 상단에는 체크아웃 날짜가 표기된다.
- 지정된 인원수는 호텔예약페이지 화면에 적용된다.

인원수 지정 페이지

- 숙박할 인원(성인, 아동)을 가운데 ‘+’, ‘-“버튼을 통해서 인원 수 조절 가능하다.
- 지정된 인원수는 호텔예약페이지 화면에 적용된다.

호텔 상세보기 페이지

- 날짜, 인원수, 호텔 선택이 완료되면 나타내며, 호텔에 대한 자세한 정보를 소개하는 페이지이다.
- 사진, 객실선택, 위치, 시설, 후기로 이루어진다.
- 객실선택에서는 더블룸, 트윈룸(기본은 더블룸)을 선택할 수 있으며, 룸 하나당 최대 2인 까지 사용가능하다.
- 위치에서는 현재 호텔의 주소를 제공하며, 지도에 호텔 위치를 나타내어 준다. 지도는 카카오API를 이용한다.
- 시설에서는 호텔 등급에 따라 차등을 둔다.
- 후기에서는 해당 호텔을 이용한 소비자의 평가를 글로 남길 수 있으며, 이는 모든 이용자에게 공개로 제공된다.
- 객실 선택에서 선택된 룸의 개수에 따라 결제금액이 정해지며, 결제 버튼을 누르면 결제창으로 이동한다.

결제 페이지

- 결제창은 네이버페이API를 이용하여 사용한다.
- 결제 창에서는 포인트, 카드, 무통장입금 선택 여부를 알려주며, 선택된 결제 방법에 따라 결제가 진행된다.
- 결제가 완료가 되면, 예약 완료 페이지가 실행된다.
- 결제 오류 및 실패시에는 결제 실패 페이지 알람이 뜨며, 알람창에서 확인 버튼을 누르면 호텔 상세보기 페이지로 돌아간다.

예약완료페이지

- 호텔 예약이 성공적으로 완료되었다는 문구와 함께 호텔 예약 상세 내역을 제공한다.
```
